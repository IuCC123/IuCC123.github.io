GrandmaAI API Documentation

This is the API documentation for GrandmaAI. You can use this to talk to the AI chatbot.

Getting Started

To get started, you simply need to send a GET request to the API with the following data:

  • id - The chat id of the chat you want to talk to and save chat history to
  • msg - The message you want to send

Example request: https://ai.grandma-does.tech/?id=1&msg=Hey!

Example response:

{"response":{"response":"Hello! How can I assist you today?"},"chat":{"messages":[{"role":"system","content":"[system prompt, including chat history]"},{"role":"user","content":"Hey!"},{"role":"bot","content":{"response":"Hello! How can I assist you today?"}}],"id":"[chat id]"}}
